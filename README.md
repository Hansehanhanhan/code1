# AI Agent 商家运营助手项目

这是一个综合性的 AI Agent 演示项目，包含两个版本的后端实现（原生版与 LangChain 版）以及一个统一的组件化 UI 前端。项目旨在展示如何利用大模型能力构建面向商家运营场景（如流量分析、广告诊断、库存检查等）的高度自动化工作流。

## 核心功能

- **智能规划**：根据商家反馈的问题（如“最近流量下滑严重”）自动拆解任务。
- **工具调用**：自动化执行流量分析、竞品对比、库存预警等模拟工具。
- **结果校验**：自动验证分析结果的逻辑性，并给出最终修复建议。
- **现代 UI**：基于 Next.js 和 Tailwind CSS 的深蓝色调响应式界面。

## 快速启动

### 1. 前置要求

- **Python 3.10+** (推荐)
- **Node.js 18+**
- **DeepSeek/OpenAI API Key**

### 2. 后端启动 (以 LangChain 版为例)

```powershell
cd langchain-agent
# 创建并激活环境
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 安装依赖
pip install -r requirements.txt

# 配置环境变量 (.env)
echo "OPENAI_API_KEY=your_key" > .env
echo "OPENAI_BASE_URL=https://api.deepseek.com" >> .env

# 启动 (默认端口 8000)
$env:PYTHONPATH="."
python backend/main.py
```

### 3. 前端启动

```powershell
cd code1/frontend
npm install
npm run dev
```
访问地址: `http://localhost:3000`
