# 商品转化提升 SOP

## 适用场景
- 点击正常但下单率低。
- 页面停留时长短、跳失高。

## 排查重点
1. 价格竞争力：是否处于目标价格带。
2. 详情页首屏：核心卖点、优惠、保障是否清晰。
3. 评价体系：差评关键词是否集中在关键体验点。
4. 交易门槛：运费、发货时效、售后承诺是否影响下单。

## 转化优化动作
- 标题：聚焦搜索意图词，减少无效修饰语。
- 主图：突出差异化卖点和活动权益。
- 详情页：先结论后证明，强化对比和信任背书。
- 价格策略：建立引流款-利润款-形象款的梯度。

## 指标观察窗口
- 首次改版后观察 72 小时再做下一轮调整。
- 建议结合 UV 价值、加购率、支付转化率联动评估。

## 常见误区
- 只降价不优化页面，短期有效但利润和品牌受损。
- 高频改动页面，导致算法和用户行为难以稳定。
